神雕定制版-微软KMS激活服务器

说明

本系统已经自动运行kms服务器(著名的vlmcsd开源软件),可在线激活vl版windows,office。
也可以在虚拟机运行slitaz,进行宿主windows系统的激活。
也可以在远程电脑运行slitaz,输入正确的slitaz IP,进行远程激活。
本KMS服务器远程端口是默认的1688
远程时可使用 http://本系统IP/kms.html 打开本网页.

使用

激活windows系统

在windows系统上,打开运行,输入cmd,以管理员身份打开windows的终端cmd窗口(或打开powershell)

slmgr /ipk xxxx-xxxx-xxxx-xxxx-xxxx [密钥]

slmgr /skms 192.168.x.x [slitaz系统的IP地址]

slmgr /ato

激活ofiice软件

在windows系统上,打开运行,输入cmd,以管理员身份打开windows的终端cmd窗口(或打开powershell)


找到你的office安装目录,比如C:\Program Files (x86)\Microsoft Office\Office16
64位的就是C:\Program Files\Microsoft Office\Office16
office16是office2016,office15就是2013,office14就是2010.
然后目录对的话,该目录下面应该有个OSPP.VBS。
接下来我们就cd到这个目录下面,例如:


cd C:\Program Files (x86)\Microsoft Office\Office15
或者cd C:\Program Files\Microsoft Office\Office15
cscript ospp.vbs /sethst:192.168.x.x
cscript ospp.vbs /act
